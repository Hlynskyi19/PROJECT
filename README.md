# 🍃 EcoBonus

**EcoBonus** — це сучасний веб-застосунок, створений для мотивації людей до сортування та переробки сміття. Проєкт об'єднує звичайних користувачів, пункти прийому вторсировини та бізнес (магазини-партнери) в єдину корисну екосистему.

## 🚀 Головні можливості

* **Розподіл ролей:** Система підтримує 3 типи акаунтів — Клієнти, Організації (пункти прийому) та Магазини-партнери.
* **Система еко-балів:** Вбудована логіка безпечного нарахування балів за здану сировину та їх списання.
* **Маркетплейс винагород:** Магазини можуть виставляти свої пропозиції, а користувачі — купувати їх за бали, отримуючи унікальні промокоди та QR-коди.
* **Інтерактивна карта:** Зручний пошук найближчих пунктів прийому вторсировини.
* **Сучасний UI/UX:** Адаптивний дизайн із підтримкою світлої та темної тем.

## 🛠 Технологічний стек

* **Backend:** Python, Django
* **Frontend:** HTML5/CSS3, Bootstrap 5, JavaScript
* **База даних:** SQLite (для розробки)
* **Додатково:** Генерація QR-кодів, інтеграція інтерактивних карт.
